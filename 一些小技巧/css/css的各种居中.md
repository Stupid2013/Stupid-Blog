### 图片在div中垂直居中

方法一：display：table；

```html
<div id="box">
  <span>
    <img src="./images/pic.jpg" alt="" />
  </span>
</div>
```
```css
#box{
  width: 500px;
  height: 400px;
  display: table;
  border: 1px solid #d3d3d3;
  background:#fff;
}
#box span{
  display: table-cell;
  vertical-align: middle;
}
#box img{
  display: block;
  margin: 0 auto;
}
<!--[if lte IE 7]>
  <style type="text/css">
    emmm...没试过...

<![endif]-->
```
